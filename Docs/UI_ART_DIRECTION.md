# 贵阳捉鸡麻将 UI 美术方向

## 核心语言

现代国风与贵阳地域文化融合：青绿山水、南明河、甲秀楼、贵州山地云雾作为背景层；苗族、布依族几何纹样与蜡染曲线只用于边缘和装饰层；暖金负责强调，深绿色负责长时间对局的视觉稳定。

## 层级原则

1. 动态文字、房间号、昵称、分数、按钮标签全部由 UMG 渲染，不烘焙进图片。
2. 背景中心保持低细节；文化元素限制在四周，不能遮挡手牌和操作区。
3. 面板、按钮、输入框采用透明 PNG 与九宫格拉伸；图标为独立 Image。
4. 游戏牌桌使用低噪点深绿桌面，象牙白麻将牌始终拥有最高识别优先级。
5. 所有资源均为本项目通过 AI 或程序生成，不复制商业游戏界面。

## 生产方式

- 8 张背景：内置图像生成模型，统一无文字约束，最终重采样到 2560×1440。
- 面板、按钮、控件、头像框、图标与麻将牌：`Scripts/GenerateUIAssets.py` 使用 Pillow 确定性生成。
- Unreal 导入：`Scripts/ImportUIAssets.py` 配置 UI 压缩、纹理组、Mip、流送、Style DataAsset 与 UI Material。

背景最终提示词采用 `stylized-concept`，共同约束为 16:9、中央低细节、青绿/暖金、无文字/按钮/人物/水印、无颗粒与压缩伪影；每张背景仅替换对应页面场景和构图要求。
