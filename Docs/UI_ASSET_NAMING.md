# UI 资产命名

- `T_BG_*`：全屏背景。
- `T_Panel_*_9Slice`：透明九宫格面板。
- `T_Btn_<Type>_<State>`：按钮底图，State 为 Normal/Hovered/Pressed/Disabled。
- `T_Input_*_9Slice`、`T_Checkbox_*`、`T_Toggle_*`、`T_Slider_*`：表单控件。
- `T_AvatarFrame_*`、`T_Seat_*`：头像和座位状态。
- `Icon_*`：128×128 独立图标。
- `T_Tile_<Wan|Tong|Tiao>_<01..09>`：规则索引 0–26 的麻将牌面。
- `M_UI_*` / `MI_UI_*`：UI Material 与 Material Instance。
- `DA_*`：主题、Brush、ButtonStyle、字体和注册表数据资产。

原始 PNG 位于 `SourceArt/UI`；导入后的 Unreal 包位于 `/Game/UI/Textures`、`/Game/UI/Data`、`/Game/UI/Materials`。
